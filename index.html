// Algorithme
🔬

Différence de frames

Chaque image est comparée à la précédente. Les zones de différence révèlent le mouvement. Le centroïde de ces zones traque la position du doigt.

⚡

Filtre à seuil adaptatif

Un seuil de bruit (configurable) élimine les micro-mouvements parasites — lumière, tremblement de main, variations d'exposition.

🎯

Isolation axe vertical

Seule la composante Y du vecteur de déplacement est transmise au scroller. Les mouvements horizontaux sont ignorés via un ratio de dominance.

« La meilleure interface est celle qui disparaît. Quand votre doigt contrôle l'écran sans le toucher, l'outil s'efface — seul le contenu demeure. »
// Cas d'usage
♿

Accessibilité motrice

Permet à des personnes avec mobilité réduite de naviguer sans toucher l'écran — particulièrement utile en situation de cuisine, sport, ou mains occupées.

🏃

Mode sportif

Lecture d'un programme d'entraînement ou d'une recette avec les mains gantées ou humides — zéro contact nécessaire.

🎤

Présentations & téléprompteur

Faire défiler un script sans télécommande ni assistant — juste un geste discret hors champ.

// Confidentialité

✅ Aucun stockage vidéo. Le flux est traité en mémoire vive, frame par frame. Aucune image n'est sauvegardée, transmise, ou journalisée.

✅ Traitement 100% local. Aucune API distante. Aucun cloud. L'analyse se fait entièrement sur votre appareil, hors ligne.

✅ Arrêt automatique. La caméra se coupe dès que l'application passe en arrière-plan ou que l'onglet perd le focus.

// Pour commencer

📱 Instructions

1. Appuyez sur 📷 en haut à droite
2. Accordez l'accès à la caméra
3. Placez votre doigt devant l'objectif
4. Bougez-le vers le haut ou le bas
5. La page défile automatiquement ✨
