💡

Lumière = direction

Lumière en haut de l'objectif → descend. Lumière en bas → remonte. Noir total → arrêt.

⚡

Vitesse = réglage

La vitesse de défilement est définie uniquement par le curseur vitesse. La luminosité ne l'influence pas.

🎯

Position = bouton rotatif

Réglez où se trouve la lumière sur votre objectif selon la façon dont vous tenez votre téléphone.

// Instructions

📱 Comment utiliser

1. Appuyez sur 📷 pour activer la caméra
2. Placez votre doigt sur l'objectif dorsal
3. Laissez passer la lumière d'un côté
4. Haut = descend   Bas = remonte
5. Noir total = arrêt ✨

Réglez la position dans ⚙️ selon comment vous tenez votre téléphone.
Ajustez la vitesse et la sensibilité indépendamment.
