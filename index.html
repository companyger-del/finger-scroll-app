// Comment ça marche
💡

Lumière = direction

Lumière dans la zone “descend” → défilement vers le bas. Lumière dans la zone “remonte” → défilement vers le haut. Noir quasi total → arrêt.

⚡

Vitesse indépendante

La vitesse dépend uniquement du curseur prévu pour cela. La détection lumineuse ne modifie pas la vitesse.

🎯

Position adaptable

Le cercle de réglage permet d’adapter les deux zones de lumière selon votre manière de tenir l’appareil.

« La meilleure interface est celle qui disparaît. »
// Confidentialité

✅ Aucun envoi réseau pour l’analyse vidéo. Le traitement reste local dans le navigateur. La caméra se coupe automatiquement si l’onglet passe en arrière-plan.

// Instructions

📱 Comment utiliser

1. Appuyez sur 📷 pour activer la caméra
2. Placez votre doigt sur l’objectif dorsal
3. Laissez passer la lumière d’un côté
4. Zone “descend” = descend   Zone “remonte” = remonte
5. Noir total ou quasi total = arrêt

Ajustez la vitesse, la sensibilité et la position dans ⚙️.
Double tapez dans la zone de lecture pour couper rapidement la caméra.
