💡

Doigt sur l'objectif

Laissez passer la lumière d'un côté → défilement. Noir total → arrêt. Ajustez la position dans les paramètres.

// Caméra frontale (nouveau)
👁

Clignements des yeux

Œil gauche → action gauche · Œil droit → action droite · Double clignement → pause/reprise · Clignement long → menu

😊

Expressions faciales

Sourcils froncés → mode avancé · Sourire → confirmation · Nez plissé → fermeture rapide. Activez via bord gauche.

✊

Pierre Feuille Ciseaux

Activez via bord bas. Montrez votre geste à la caméra frontale. Pierre = stop · Feuille = scroll · Ciseaux = sélection.

// Bords tactiles (nouveau)

Bord gauche — Active/désactive les expressions faciales
Bord droit — Augmente la vitesse de défilement
Bord haut — Menu principal
Bord bas — Ouvre Pierre Feuille Ciseaux

// Microphone (nouveau)
🎙

Signaux sonores discrets

Sifflement · Souffle · Claquement de langue. Aucun son de validation — retour haptique uniquement.

// Signe d'alerte (nouveau)
🚨

Alerte silencieuse

Paume ouverte → poing fermé rapidement devant la caméra. Déclenche une alerte discrète. Zéro son. Haptique uniquement.

« La meilleure interface est celle qui disparaît. »
