GESTCTRL

Mode actif

Système de contrôle gestuel opérationnel. Déplacez votre main devant la caméra frontale pour faire défiler ce contenu. Utilisez les bords de l'écran pour activer les fonctions avancées.

Clignement des yeux

Clignez de l'œil gauche pour aller à l'onglet précédent. Clignez de l'œil droit pour aller au suivant. Double clignement pour pause. Clignement long pour ouvrir le menu.

Expressions faciales

Froncez les sourcils pour activer le mode avancé. Souriez pour confirmer une action. Plissez le nez pour fermeture rapide et discrète du contenu sensible.

Signaux microphone

Sifflez doucement pour déclencher une action. Soufflez vers le micro pour une commande alternative. Claquement de langue pour validation rapide. Tout se fait en silence absolu pour l'entourage.

Pierre Feuille Ciseaux

Montrez poing fermé (pierre) pour stopper le défilement. Main ouverte (feuille) pour défilement libre. Deux doigts écartés (ciseaux) pour sélection. Activez via le bord bas.

Zones de bord

Bord gauche : activation mode expression faciale. Bord droit : vitesse de défilement. Bord haut : menu principal. Bord bas : mode Pierre Feuille Ciseaux et jeu.

Signe d'alerte

Main ouverte puis poing fermé rapidement devant la caméra dorsale déclenche silencieusement une alerte d'urgence. Aucun son — confirmation uniquement par haptique.

Accessibilité universelle

Ce système fonctionne pour tous — aveugles (clignements), sourds-muets (langue des signes + sourire + haptique), mobilité réduite (micro). Zéro son de validation. Zéro exclusion.

Identification par doigt

Caméra dorsale — objectif macro synchronisé au flash. Chaque doigt donne accès à un espace différent. La paume et ses lignes constituent un identifiant biométrique unique et non intrusif.

Version 1.0

Base opérationnelle. Testez chaque fonctionnalité et notez vos retours pour la prochaine session d'amélioration. Le système apprend de votre usage.
